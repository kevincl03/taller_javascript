Taller JavaScript - POE - Kevin Chanchi

Solución:

Punto 1.

Contador de Palabras Únicas: Cree un programa que solicite al usuario ingresar un párrafo y luego cuente cuántas palabras únicas hay en ese párrafo. Ignore las mayúsculas y minúsculas, es decir, trate "Hola" y "hola" como la misma palabra.



Punto 2.

Validación de Contraseña: Cree un programa que solicite al usuario crear una contraseña. Luego, verifique si la contraseña cumple con ciertos criterios, como tener al menos 8 caracteres, contener al menos una letra mayúscula y un número, y no contener espacios en blanco.



Punto 3.

Anagramas: Cree un programa que determine si dos cadenas ingresadas por el usuario son anagramas entre sí. Dos palabras son anagramas si contienen las mismas letras, pero en un orden diferente.



Punto 4.

Reemplazo de Caracteres: Cree un programa que reemplace todas las vocales en una cadena con un carácter específico, por ejemplo, 'a' por '*'.



Punto 5.

Buscar en una Cadena: Pida al usuario una cadena de texto y una palabra clave, y luego muestre si la palabra clave aparece en la cadena.



Punto 6.

Contador de Palabras: Cree un programa que cuente cuántas palabras hay en una frase ingresada por el usuario.



Punto 7.

Inversión de Cadena: Cree una función que invierta una cadena de texto dada. Luego, use esta función para invertir una cadena ingresada por el usuario.



Punto 8.

Cadenas y Arrays: Cree un array de cadenas que contenga nombres de frutas. Pida al usuario que ingrese un nombre de fruta y verifique si esa fruta está en el array.



Punto 9.

Mayúsculas y Minúsculas: Pida al usuario que ingrese una cadena y luego muestre la misma cadena con todas las letras en mayúsculas y todas las letras en minúsculas.



Punto 10.

Longitud de Palabras: Cree un programa que tome una oración ingresada por el usuario y muestre la longitud de cada palabra en esa oración.



Punto 11.

Duplicar Elementos en un Array: Cree un programa que tome un array de números enteros y utilice la función map para duplicar cada elemento en el array. Luego, muestre el array resultante.

numerosEnteros = [2, 4, 1, 6, 8, 5, 12, 16, 22, 34, 50, 42]

Punto 12.

Mayúsculas a Minúsculas y Viceversa: Cree un programa que tome una lista de palabras en un array y utilice la función map para convertir todas las letras en mayúsculas a minúsculas y viceversa en cada palabra. Luego, muestre la lista de palabras modificada.
